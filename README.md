# PRShield
GitHubでプルリクエストを作成する際、マージ先ブランチを確認するのを強制するChrome拡張機能です。
チェックボックスをチェックしなければプルリクエストの作成ができないようになっています。

![image](https://github.com/user-attachments/assets/01fae708-d33b-4c62-9622-af9c08ccd569)

## 使い方
Chromeウェブストア審査中のため、現時点で利用したい場合は下記手順に従ってインストールしてください。
1. このリポジトリをクローンする
2. `chrome://extensions/`にアクセスし、Chromeの拡張機能を開く
3. 右上のデベロッパーモードをオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、クローンしたリポジトリを選択する
